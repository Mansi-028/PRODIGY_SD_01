## 📖 About The Project

A simple **Temperature Converter** that converts a temperature value from one unit (Celsius, Fahrenheit, or Kelvin) to the other two units.

This project is part of my **Software Development Internship at Prodigy InfoTech**. It comes in **two versions**:

| Version | Tech | Folder |
| --- | --- | --- |
| 🌐 Web App | HTML, CSS, JavaScript | `temperature-converter-web/` |
| 🖥️ Desktop App | Java Swing | `temperature-converter-java/` |

---

## ✨ Features

- ✅ Convert between **Celsius (°C)**, **Fahrenheit (°F)** and **Kelvin (K)**
- ✅ Input validation (no empty / invalid numbers)
- ✅ Checks for **absolute zero** (temperature can't go below 0 K)
- ✅ Clean and responsive UI
- ✅ Reset button to clear everything
- ✅ Available as both a Web App and a Java Desktop App

---

## 🧮 Conversion Formulas Used

```
°F = (°C × 9/5) + 32
°C = (°F − 32) × 5/9
K  = °C + 273.15
°C = K − 273.15
```

## 🚀 Getting Started

### 🌐 Web Version

```bash
cd temperature-converter-web
# Just open index.html in a browser
```

Or host it on **GitHub Pages** for a live demo 🌍

### 🖥️ Java Version

> Requires **JDK 8+** installed

```bash
cd temperature-converter-java
javac TemperatureConverter.java
java TemperatureConverter
```
