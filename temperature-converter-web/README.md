# Temperature Converter - Web Version

Task 01 of Prodigy InfoTech Software Development Internship.
Built with plain HTML, CSS and JavaScript (no frameworks).

**Author:** Mansi (B.Tech)

## How to run

Just open `index.html` in any web browser. That's it.

## Files

- `index.html` - the page structure
- `style.css`  - styling
- `script.js`  - conversion logic

## Formulas used

- C to F: `F = (C * 9/5) + 32`
- C to K: `K = C + 273.15`
- F to C: `C = (F - 32) * 5/9`
- K to C: `C = K - 273.15`
